Tentang Saya / About Me

Hai! Saya Jason Aditya Tjiong, lulusan SMK TI Bali Global Singaraja, dan saat ini sedang menempuh pendidikan sebagai mahasiswa di INSTIKI Bali.

Saya memiliki hobi memotret dan bermain game. Saya juga sangat menikmati kegiatan traveling sambil hunting foto. Peralatan yang saya gunakan untuk memotret yaitu kamera Sony A7II dengan dua lensa andalan saya, Lensa 85mm f/1.8 dan Lensa 24–105mm f/4. Selain kamera, saya juga sering memotret menggunakan ponsel, dan hasilnya pun tidak kalah bagus!

Banyak teman yang ingin ikut memotret, tetapi merasa terhalang karena tidak memiliki kamera profesional atau lensa mahal. Padahal, ponsel zaman sekarang sudah sangat mumpuni untuk fotografi. Yang penting adalah memahami logika kerja kamera dan belajar komposisi gambar.

Apakah saya sudah profesional? Belum. Saya masih belajar. Saya belum ahli dalam memotret model atau mengatur pose, tapi saya terus berusaha berkembang dan belajar dari setiap pengalaman.

📸 Fotografi bukan tentang alat, tapi tentang rasa dan cerita.

Dunia Game Saya

Selain fotografi, saya juga memiliki ketertarikan besar di dunia game. Saya sering bermain di waktu senggang, dan kalau ditanya saya main game apa, jawabannya adalah: banyak! Saya menikmati berbagai genre, mulai dari horor, RPG, FPS, hingga simulasi dan petualangan. Beberapa judul yang sering saya mainkan antara lain Dead by Daylight, Minecraft, Roblox, Left 4 Dead 2, Project Zomboid, Stardew Valley, Terraria, PUBG, Valorant, CS:GO, Apex Legends, Mobile Legends, dan masih banyak lagi.

Bermain game bukan hanya soal hiburan, tapi juga cara saya bersosialisasi, belajar strategi, dan mengisi waktu dengan hal yang menyenangkan.

Berbicara tentang game, saya juga mengelola server Minecraft Java Edition yang saya host sendiri. Server ini menjadi tempat bagi saya dan teman-teman untuk bermain bersama, bereksperimen dengan mod, dan menikmati dunia Minecraft dengan cara yang unik. Jika kamu tertarik untuk bergabung, jangan ragu untuk menghubungi saya atau bergabung ke server Discord kami. Bermain bersama selalu lebih seru, bukan?
